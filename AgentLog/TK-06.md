# TK-06 横展開

## ticket id and title
- TK-06 横展開

## goal
- トップ基準の見出し・記事カード・タグ導線を、`single.php`、`page.php`、`category.php`、`tag.php`、`search.php`、`sidebar.php` に横展開しつつ、各テンプレートの役割は維持する。

## 変更内容
- `inc/gakuson-content-helpers.php`
- 下層で共通に使う `gakuson_get_section_title_markup()`、`gakuson_get_article_card_markup()`、`gakuson_get_tag_directory_markup()` を追加した。
- `single.php` / `page.php`
- 本文まわりは維持しつつ、関連記事とタグ一覧をトップ基準の section heading / article card / tag list に差し替えた。
- `category.php` / `tag.php`
- 一覧の見出しをトップ基準の section heading に変更し、記事一覧を `article_item` 系カードへ揃え、ページングを追加した。
- `search.php`
- 作業ツリー上で欠けていた検索結果テンプレートを復元し、現在条件、再検索フォーム、結果カード、0件メッセージ、ページングをまとめた。
- `sidebar.php`
- 人気記事と外部導線を section ごとに整理し、見出しトーンをトップ基準へ寄せた。順位表現そのものは `1st.TIPS` など既存方針を残した。
- `smacss/module/module.scss`
- トップ以外でも使う section heading、article card、tag list のスタイルを module 層へ移した。
- `smacss/module/sidebar.scss`
- サイドバー section の余白と外部導線カードのトーンを調整した。
- `smacss/page/page-category.scss`
- archive / search 用の導入文、検索条件、再検索フォーム、empty state、pagination の受け皿を追加し、旧 `.article_title` 競合を解消した。
- `smacss/page/page-post.scss` / `smacss/page/page-fixed.scss`
- 関連記事セクションと下部タグ導線の余白を追加した。
- `smacss/page/page-top.scss`
- module へ移した共通スタイルをトップ専用ファイルから外した。
- `smacss/main/main-top.css` / `smacss/main/main-post.css` / `smacss/main/main-fixed.css` / `smacss/main/main-category-tag.css`
- Sass ビルドで再生成した。

## 判断理由
- TK-06 の対象テンプレートが多く、`article_item` と `section_TitleConteiner` の横展開を template ごとにコピペすると保守が重くなるため、WordPress theme の helper と module 層に薄く寄せた。
- `single.php` と `page.php` は「本文構造は維持」が優先なので、本文本体ではなく関連記事・タグ導線だけをトップ基準に寄せた。
- `category.php` / `tag.php` / `search.php` は一覧ページとしての役割が近いため、見出し・カード・ページングのルールだけ共通化し、テンプレート統合はしていない。
- `sidebar.php` は narrow layout なので、トップの横長 `article_item__popu` までは持ち込まず、見出しトーンと広告カードの見せ方だけを寄せた。
- `search.php` は現状の workspace で欠けており TK-06 の対象を満たせなかったため、依存 ticket の前提を壊さない最小構成で復元した。
