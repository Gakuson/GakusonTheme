# TK-05 トップカルーセル

## ticket id and title
- TK-05 トップカルーセル

## goal
- `featured` タグ付き記事を使ったトップ訴求エリアを `front-page.php` に追加し、0件/1件/複数件の表示条件と、説明文・カテゴリ・タグ・CTA・スライド操作を揃える。

## 変更内容
- `inc/gakuson-content-helpers.php`
- `gakuson_get_post_tag_names()` に `exclude_slugs` オプションを追加し、表示側だけ `featured` タグを除外できるようにした。
- `front-page.php`
- トップ先頭に注目記事カルーセルを追加し、`gakuson_get_featured_posts()` を使って最大5件の `featured` 記事を表示するようにした。
- 0件時は section 自体を出さず、1件時は静的ヒーロー、2件以上では前へ/次へ・ドット・現在位置表示を出す構成にした。
- 各スライドでアイキャッチ、タイトル、抜粋、カテゴリ、`featured` を除外したタグ、CTA を表示するようにした。
- `js/script.js`
- 既存 jQuery 内にトップカルーセル初期化を追加し、前後移動、ドット移動、現在位置の同期、`hidden` / `aria-hidden` の切り替えを実装した。
- `smacss/page/page-top.scss`
- トップ専用のカルーセルスタイルを追加し、PC/SP のレイアウト、操作UI、チップ、CTA を調整した。

## 判断理由
- `featured` 記事取得と抜粋ルールは TK-01 の helper をそのまま使い、TK-05 ではトップ描画と操作UIに責務を絞った。
- タグ除外は endpoint payload 全体を先に変えず、表示用途だけ `exclude_slugs` で差し引けるようにして、TK-08 の仕様判断を残せる形にした。
- スライダーは既存 jQuery に寄せつつ、非表示スライドへフォーカスが残らないよう `hidden` と `aria-hidden` を同期する方式にした。
