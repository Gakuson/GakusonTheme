# TK-01 基盤整備

## Goal
- 後続チケットの前提になる helper の置き場と責務を先に固定し、Sass の出力導線を現行テーマ構成へ合わせる。

## Implementation Summary
- `functions.php` から `inc/gakuson-content-helpers.php` を読み込む構成に変更した。
- `inc/gakuson-content-helpers.php` を追加し、featured 記事取得、カルーセル response 整形、タグクラウド整形、transient cache helper と無効化 hook をまとめた。
- `package.json` の Sass scripts を `smacss/main` 出力に合わせて更新し、`build:css` と `watch:css` を整理した。

## Touched Files
- `functions.php`
- `inc/gakuson-content-helpers.php`
- `package.json`

## Review Notes / Risks
- `gakuson_get_carousel_cache_ttl()` の既定値 5 分は仮置きで、最終値は `TK-08` までに要確定。
- カルーセル payload の `tags[]` に `featured` 自体を含めるかは未決。
- サムネイル未設定時の fallback 画像方針は未確定。
- この環境では `php` コマンドが見つからず lint 未実施、`npm run build:css` は sandbox 由来の `EPERM` で未完了。
